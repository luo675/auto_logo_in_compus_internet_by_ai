# AI提示词模板 - 校园网自动登录工具

## 项目概述

开发一个Windows自动登录工具，解决以下问题：
1. 每次开机需要手动登录校园网
2. 有时忘记关闭VPN导致网络配置异常，需要火绒断网修复
3. 需要自动化整个过程：检测网络 → 修复问题 → 自动登录

## 技术栈要求

- **编程语言**: Python 3.9+
- **GUI自动化**: pyautogui + pywinauto
- **系统操作**: subprocess + psutil
- **网络检测**: requests + ping3
- **通知**: win10toast
- **打包**: PyInstaller

## 核心需求

### 1. 开机自启动
- 系统启动后延迟30秒运行
- 支持Windows计划任务或注册表启动

### 2. 网页自动关闭
- 自动检测并关闭校园网登录网页
- 支持多种浏览器 (Chrome, Edge, Firefox, IE)
- 安全关闭，不影响其他浏览器标签页

### 3. 网络状态检测
- Ping测试网络连通性 (8.8.8.8)
- 检测VPN残留的网络配置问题
- 判断是否需要网络修复

### 4. 自动修复功能
- 调用火绒断网修复工具
- 路径: `D:\APP\huorongsecurity\Huorong\Sysdiag`
- 需要查找具体可执行文件名 (可能是 hrfix.exe, hrnetfix.exe, netset.exe)
- 支持静默模式运行

### 5. Dr.COM客户端自动化
- 客户端路径: `C:\Users\luo\Desktop\APP\DrClient.exe`
- 自动启动客户端
- 模拟GUI操作输入账号密码
- 点击登录按钮
- 检测登录状态

### 6. 错误处理和日志
- 详细的运行日志记录
- Windows toast通知
- 重试机制 (最多3次)
- 优雅的错误恢复
