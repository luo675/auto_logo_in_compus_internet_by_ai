# 校园网自动登录工具 - 计划总结

## 问题分析

基于 `goal.md` 的分析，核心问题如下：
1. **开机手动登录**: 每次开机需要手动登录校园网
2. **网页自动弹出**: 开机时会弹出校园网登录网页，需要手动关闭
3. **VPN残留问题**: 忘记关闭VPN导致网络配置异常
4. **修复流程繁琐**: 需要手动打开火绒断网修复工具

## 解决方案概述

开发一个自动化工具，实现：
1. **开机自启动**: 系统启动后自动运行
2. **网页自动关闭**: 自动检测并关闭校园网登录网页
3. **智能检测**: 自动检测网络状态和VPN残留问题
4. **自动修复**: 调用火绒工具修复网络问题
5. **自动登录**: 模拟GUI操作登录Dr.COM客户端
6. **错误处理**: 完善的日志、通知和重试机制

## 技术要点

### 核心模块
1. **配置管理**: 加密存储账号密码，支持Windows凭据管理器
2. **网络检测**: Ping + HTTP测试，VPN残留检测
3. **修复工具**: 自动查找并调用火绒断网修复工具
4. **GUI自动化**: 使用pywinauto/pyautogui操作Dr.COM客户端
5. **日志通知**: 文件日志 + Windows toast通知
